1) После того как Вы скачали мой репозиторий, 
    Вам потребуется установить node.js если эта полезная штука у Вас еще не установлена.
    (это нужно для того, чтобы работать с пакетным менеджром npm)
 
2) Далее вам нужно будет перейти в папку с пректом и в ней запустить консоль / 
    либо перейти в эту папку через консоль заранее (разницы нет)
 
3) Когда вы будете находится в папке с проектом, 
    в консоли понадобиться написать команду: ' npm install ' 
    - эта команда загрузит все необходимые модули для запуска приложения. 
    (Это может занять несколько минут)
 
4) После того как все будет установлено вам нужно прописать следующую команду: ' npm run dev '
    - проект соберется и вы сможете получить к нему доступ через браузер по адресу http://localhost:3000/
    ( также это будет написано и в консоли )
